# Context Pack: Frontend

## Purpose
Provides frontend development conventions, component patterns, and UI standards for all AI agents working on client-side code generation.

## Files

| File | Description | Load When |
|------|-------------|-----------|
| `component_patterns.md` | React component architecture, composition | Any frontend component work |
| `state_management.md` | Global/local state, server state, caching | State-related changes |
| `forms_and_validation.md` | Form handling, validation, error display | Form implementation |
| `routing_and_navigation.md` | Route structure, guards, navigation patterns | Adding pages/routes |
| `real_time.md` | WebSocket, SSE, live updates | Real-time features |
| `data_visualization.md` | Charts, DAG visualization, metrics display | Dashboard/analytics work |
| `accessibility.md` | ARIA, keyboard nav, screen readers | All UI work (always load) |
