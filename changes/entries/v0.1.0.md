**Date:** 2025-02-20

**Author:** System Architect

## Summary
First version of the AI Software Factory prompt repository. Establishes the complete project structure, all module definitions, shared conventions, and context packs.

## What's Included

### Shared Layer (6 files)
- `backend.md` — Backend architecture (FastAPI + Node.js hybrid, PostgreSQL, Redis)
- `frontend.md` — Frontend architecture (React 19, TypeScript, Tailwind, shadcn/ui)
- `design.md` — Design system (colors, typography, spacing, components)
- `app.md` — Application config, context loading formula, module registry
- `personas.md` — User roles (Admin, Team Lead, Developer, Viewer) with permissions
- `versioning.md` — Versioning strategy (SemVer, API, DB, file-level)

### Features (8 modules)
Each module includes: feature_brief, back/ (data_model, api_endpoints), front/ (pages), workflows/

1. **command_ui** — Human-facing dashboard for task management
2. **planning_service** — AI-powered goal decomposition into plans
3. **approval_gateway** — Human review and approval checkpoints
4. **prompt_factory** — Repository building and delta management
5. **orchestrator** — DAG execution engine with state management
6. **agent_runtime** — AI agent execution environment
7. **knowledge_base** — Vector search and code embeddings
8. **feedback_engine** — Signal collection and performance improvement

### Context Packs (4 packs, 18 files)
- **backend/** — API conventions, database patterns, service layer, auth, testing, error handling
- **frontend/** — Component patterns, state management, real-time, data visualization
- **orchestration/** — Execution patterns, agent coordination, state transitions, error recovery, resource management
- **memory/** — Decisions log, conventions, glossary, known issues, tech debt
